# ☕ KleCoffee - Sistema de Gerenciamento

Sistema completo para gerenciamento da cafeteria KleCoffee com dashboard, controle de máquinas, clientes, eventos, estoque e pagamentos.

## 🚀 Funcionalidades

- **Dashboard**: Visão geral do negócio com gráficos e estatísticas
- **Máquinas**: Controle de equipamentos e manutenções
- **Clientes**: Cadastro e histórico de clientes
- **Eventos**: Agendamento e acompanhamento de eventos
- **Estoque**: Controle de produtos e alertas de estoque baixo
- **Pagamentos**: Registro de vendas e formas de pagamento

## 🛠️ Tecnologias

- **Backend**: Node.js + Express
- **Banco de Dados**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript Vanilla
- **Gráficos**: Chart.js
- **Ícones**: Font Awesome

## 📋 Pré-requisitos

- Node.js (v18 ou superior)
- MySQL (v8.0 ou superior)
- NPM (v9 ou superior)
